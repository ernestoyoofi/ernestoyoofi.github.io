Apa Itu AMP HTML Website ?

AMP Adalah Singkatan Dari "Accelerated Mobile Page", "Accelerated Mobile Page" bertujuan agar page dibuka para pengguna Moblie / Handphone agar lebih cepat dalam memuat halaman website

Fitur ini sudah banyak dipakai pada website yang membutuhkan memuat halaman yang cukup lama

Fitur ini dikembangkan olehGoogle,Google AMP merupakan proyek idealis dari Google.

HTML AMP berfungsi untuk meminimalisir penggunaan tag JavaScript serta mendorong akses prioritas pengguna perangkat mobile. HTML AMP bekerja untuk mengecilkan ukuran HTML, menyesuaikan gambar, dan CSS, hingga inline CSS, serta berfungsi mempercepat akses dalam metode asynchronus pada perangkat ponsel pintar. AMP CDN berfungsi menyimpan data dalam bentukCache.

Namun seiring berjalanya waktu dan kemajuan teknologi yang sangat pesat, kini tidak hanya google yang memanfaatkan amp ini, sekarang sosial media paling banyak penggunanyaFacebookjuga telah menerbitkan amp yang juga mempunyai fungsi yang sama seperti google, yaitu untuk mempercepat waktu loading yang dengan itu akan mengurangi berbagai fungsi kompleks pada javascript dan html yang memberatkan, anda dapat membaca pengaturan teknis untuk AMP pada website resmi mereka.

Kekurangan Fitur Tersebut Apa Saja ?

1. Tidak bisa bebas dalam menggunakan script, script yang dipakai hanyalahJavaScript

2. Keseluruhan penggunaan lebih sulit

3. Sulit membedakan website aslinya

4. Penggunaan sulit dalam memahami fiturnya

Kelebihan Fitur Tersebut Apa Saja ?

1. Meningkatkan pengalaman pengguna dalam menelusuri website

2. Memperingankan beban kerja pada komputer server

3. Loding page yang super cepat

4. Meringankan mobilitas pengguna mobile / handphone

5. Mempercepat keluar masuknya halaman website
